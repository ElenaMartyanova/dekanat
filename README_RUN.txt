# Virtual Dean

Информационный сервис виртуального деканата для поддержки студентов, в том числе иностранных обучающихся.

Проект реализован на FastAPI и использует PostgreSQL для хранения базы вопросов и ответов, необработанных обращений пользователей, обратной связи и данных администратора.

## Основные возможности

- поиск ответов по базе вопросов и ответов;
- поддержка русского, английского и китайского языков;
- фильтрация поиска по тематическим разделам;
- сохранение необработанных вопросов пользователей;
- сбор обратной связи и оценки качества работы сервиса;
- административная панель;
- добавление, редактирование и удаление вопросов и ответов;
- выгрузка отчетов в Excel;
- REST API;
- Swagger-документация.

## Технологии

- Python
- FastAPI
- PostgreSQL
- Jinja2
- HTML/CSS
- RapidFuzz
- OpenPyXL
- Swagger / OpenAPI
